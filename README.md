# Arduino Custom Data Station

## Display Plain Text Data

### Run
* `git clone https://github.com/DevTomek-pl/Arduino-Custom-Data-Station.git`
* `cd ./Arduino-Custom-Data-Station`
* `git checkout display-plain-text-data`
* `npm install`
* `npm start`
